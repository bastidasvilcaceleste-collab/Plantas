¿Cómo funciona?

Tres pasos simples para obtener el diagnóstico de tu cultivo

1
Selecciona tu cultivo

Elige entre papa, tomate o maíz. Cada cultivo tiene un modelo entrenado específicamente para detectar sus enfermedades.

2
Sube una foto de la hoja

Toma una foto clara de la hoja con buena iluminación sobre un fondo uniforme para obtener el mejor resultado.

3
Obtén el diagnóstico

Recibe el resultado con el estado de salud, nivel de confianza, recomendaciones de tratamiento y consejos de cultivo.
